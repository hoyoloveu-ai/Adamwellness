# Adam Wellness

Personlig träning — Next.js + Tailwind, svart/vit single-page site.

## Lokalt

```bash
npm install
npm run dev
```

Öppna [http://localhost:3000](http://localhost:3000).

## Live

GitHub Pages: https://hoyoloveu-ai.github.io/Adamwellness/

Varje push till `main` bygger och publicerar automatiskt.

## Byta foton

Sektionerna använder platshållare. Lägg bilder i `public/images/` och uppdatera komponenterna till `next/image` när du är redo.

## Bokning / betalning

Knapparna är styled platshållare — koppla bokningssystem och Stripe/Swish senare.
